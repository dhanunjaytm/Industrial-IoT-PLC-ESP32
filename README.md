# Industrial-IoT-PLC-ESP32
Industrial IoT monitoring using ESP32 and Siemens S7-1200 PLC with Firebase cloud integration

Project Overview

This project was developed to enable real-time industrial data monitoring by integrating Siemens S7-1200 PLC with ESP32 for cloud-based access.

The PLC collects industrial process data, which is transferred through Ethernet TCP/IP communication to the ESP32. The ESP32 then sends this data to Firebase Cloud for remote monitoring and analysis.

The main goal of this project was to improve industrial visibility, allow remote access to machine data, and support better monitoring of industrial operations in real time.

---

Technologies Used

* Siemens S7-1200 PLC
* ESP32
* Ethernet TCP/IP Communication
* Firebase Cloud
* Embedded C
* Industrial IoT
